# ACTIVIDAD-IA-ANDRES-JULIAN-ROLON-IBERO

Ayúdame a crear un dashboard para una tienda ficticia llamada TrendGear. Necesito el código en 3 archivos separados: HTML, CSS y JS (no todo junto, por favor).
Primero necesito un dataset sintético con estas 11 columnas exactas: Customer ID, Name, Email, Product Purchased, Purchase Date, Amount Spent, Age, City, Payment Method, Last Login Date, Membership Status. Que sean como 30-40 clientes inventados, con edades entre 13 y 100, montos en $0 o más, fechas bien formateadas (que la fecha de compra no sea después del último login). Dame también un JSON con esa estructura para poder importarlo a la siguiente Firebase: https://actividad-ia-default-rtdb.firebaseio.com/
